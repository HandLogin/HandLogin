Introducing a (somewhat) restful API

כדי להשתמש במערכת בפרוייקט שלכם:

<script type="text/javascript" src="http://handlogin.herokuapp.com/scripts/api.js"></script>
<script> // Your app name const APP_NAME = "שם האתר שלכם"; // פונקצייה שמקבלת את התוצאות function resultHere(tag, status, result){ // tag - תיאור הפעולה (לדוגמה מצא-שם-חיה) // status - האם הפעולה הצליחה // result - תוצאת הפעולה או שגיאה } </script>

כדי ליצור משתמש חדש:

// name password tag signUp("name", "password", "הירשמות");

כדי להיכנס עם המשתמש:

// name password tag signIn("name", "password", "כניסת-משתמש");

כדי לבדוק אם המשתמש מחובר:

// tag validate("בדיקת-משתמש");

כדי לכתוב למסד נתונים של המשתמש:

// dataName dataValue tag setData("pet name", "Shocko", "שינוי-שם-חיה");

כדי לקרוא ממסד נתונים של משתמש:

// dataName tag getData("pet name", "בדיקת-שם-חיה"); // Shocko
